## Adding YACL as a dependency

Firstly, you'll need to add the maven repository to your `build.gradle` file:

<CodeTabs>

```groovy !!tabs Groovy DSL (build.gradle)
repositories {
    maven {
        name "Xander's Maven"
        url "https://maven.isxander.dev/releases"
    }
}
```

```kotlin !!tabs Kotlin DSL (build.gradle.kts)
repositories {
    maven("https://maven.isxander.dev/releases") {
        name = "Xander's Maven"
    }
}
```

</CodeTabs>

| Minecraft Version  | Mod Loader                 | YACL version            |
| ------------------ | -------------------------- | ----------------------- |
| `1.21.2`, `1.21.3` | Fabric                     | `3.6.1+1.21.2-fabric`   |
| `1.21.2`, `1.21.3` | NeoForge                   | `3.6.1+1.21.2-neoforge` |
| `1.21`,`1.21.1`    | Fabric                     | `3.6.1+1.21-fabric`     |
| `1.21`, `1.21.1`   | NeoForge                   | `3.6.1+1.21-neoforge`   |
| `1.20.6`           | Fabric                     | `3.6.1+1.20.6-fabric`   |
| `1.20.6`           | NeoForge                   | `3.6.1+1.20.6-neoforge` |
| `1.20.4`           | Fabric                     | `3.6.1+1.20.4-fabric`   |
| `1.20.4`           | NeoForge                   | `3.6.1+1.20.4-neoforge` |
| `1.20.1`           | Fabric                     | `3.6.1+1.20.1-fabric`   |
| `1.20.1`           | Minecraft Forge (LexForge) | `3.6.1+1.20.1-forge`    |

Place the version of YACL you want to use in your `gradle.properties` file:

```properties
yacl_version=...
```

This next part will differ, depending on whether you're using Forge, NeoForge or Fabric.

You'll need to add the following dependency to your `build.gradle` file:

<CodeTabs>

```groovy !!tabs Fabric Loom
dependencies {
    modImplementation "dev.isxander:yet-another-config-lib:${project.yacl_version}"
}
```

```groovy !!tabs ForgeGradle
dependencies {
    compileOnly fg.deobf("dev.isxander:yet-another-config-lib:${project.yacl_version}")
}
```

</CodeTabs>

## Marking YACL as a dependency

You'll need to tell your users that your mod depends on YACL, you can do this by adding the following to your mod's `fabric.mod.json`, `mods.toml` or `neoforge.mods.toml`

<CodeTabs>

```json !!tabs fabric.mod.json (Fabric)
"depends": {
  "yet_another_config_lib_v3": ">=YACL_VERSION_HERE"
}
```

```toml !!tabs neoforge.mods.toml (NeoForge)
[[dependencies.your_mod_id]]
modId = "yet_another_config_lib_v3"
mandatory = true
versionRange = "[YACL_VERSION_HERE,)"
ordering = "NONE"
side = "CLIENT"
```

```toml !!tabs mods.toml (LexForge)
[[dependencies.your_mod_id]]
modId = "yet_another_config_lib_v3"
mandatory = true
versionRange = "[YACL_VERSION_HERE,)"
ordering = "NONE"
side = "CLIENT"
```

</CodeTabs>
